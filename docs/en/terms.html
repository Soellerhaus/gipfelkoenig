1. Scope

These Terms of Service apply to the use of the web app "Bergkönig" (bergkoenig.app), operated by Claudio Scheiwiller, Riezlern, Austria.

2. Description of Service

Bergkönig is a free game where users collect mountain peaks in the Alps and earn points. Summit detection works either automatically via an optional connection with Strava, Garmin Connect, or Suunto, or manually via GPS check-in.

3. Registration

Registration with an email address and password is required to use the service. By registering, you confirm:

  • That you have read the Privacy Policy
  • That you understand Bergkönig is a game and not a safety tool
  • That your username and points will appear publicly on the leaderboard

4. Strava Integration

Connecting with Strava is optional. Bergkönig is not a Strava app — Strava serves only as an optional data source for GPS activities.

  • Strava connection is only offered in Settings
  • Only GPS coordinates and elevation data are processed
  • GPS tracks are not stored
  • The connection can be disconnected at any time
  • Upon disconnection, all Strava tokens are deleted and deauthorized at Strava

Bergkönig is not operated or endorsed by Strava, Inc. Strava® is a registered trademark of Strava, Inc.

5. Safety Notice

Bergkönig is a game and NOT a safety tool.

Users are obligated to:

  • Independently inform themselves about alpine hazards
  • Check the current avalanche report at lawinen.report
  • Only undertake mountain tours with appropriate experience and equipment
  • Never take irresponsible risks because of game points

At avalanche danger level 3 or higher, Bergkönig automatically does not award points. However, this does not replace your own assessment.

6. Points and Leaderboard

Points are awarded according to a fixed system (elevation gain, first ascents, combos). The operator reserves the right to adjust the points system. There is no entitlement to cash out or exchange points.
